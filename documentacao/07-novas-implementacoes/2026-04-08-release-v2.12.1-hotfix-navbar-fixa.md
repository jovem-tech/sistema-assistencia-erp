# Release v2.12.1 - Hotfix de navbar fixa no ERP web

Data: 08/04/2026

## Objetivo

Publicar o ajuste visual da navbar superior fixa no ERP web, corrigindo a falta de sincronizacao no ambiente VPS apos a release `v2.12.0`.

## Escopo publicado

- ERP oficial `2.12.1`
- app mobile/PWA `0.4.2` (sem alteracoes nesta release)
- ajuste de layout apenas no frontend web

## Arquivos tecnicos

- `public/assets/css/estilo.css`
- `app/Config/SystemRelease.php`

## Ajustes aplicados

- `.top-navbar` com `position: fixed`, `top: 0` e z-index operacional de header.
- compensacao da area principal via `padding-top: var(--navbar-height)` em `.main-content`.
- alinhamento do `left` da navbar para:
  - `var(--sidebar-width)` com sidebar aberta;
  - `var(--sidebar-collapsed)` com sidebar recolhida;
  - `0` em viewport mobile.
- ajuste de impressao para remover compensacao superior (`padding-top: 0 !important`).

## Regras operacionais de deploy

- sem sincronizacao de dados ficticios de teste;
- sem envio de `public/uploads/`;
- preservacao de `.env` e `writable/` na VPS.
