# Login e Instalacao

## Login

O usuario entra com as mesmas credenciais do ERP. Nao existe base separada de usuarios para o app.

## Requisitos para instalacao

- HTTPS valido em producao;
- navegador compativel com PWA;
- no iPhone, instalacao manual via Safari;
- para push no iPhone, abertura do app pela Tela de Inicio.

## Fluxo padrao

1. abrir a URL oficial do app;
2. efetuar login;
3. instalar o PWA quando o navegador permitir;
4. liberar notificacoes quando o ambiente estiver apto.

## Indicadores importantes

- versao do app;
- compatibilidade minima com o ERP;
- estado de permissao de notificacao;
- estado de instalacao do PWA.
