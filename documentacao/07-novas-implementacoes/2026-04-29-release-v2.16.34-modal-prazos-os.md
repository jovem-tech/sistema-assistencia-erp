# Release v2.16.34 - motivo e autorizacao no modal de prazos da OS

Data: 29/04/2026
Versao ERP: 2.16.34

## Objetivo

Corrigir o fluxo de atualizacao rapida de prazos na listagem de ordens de servico.

## Entregas

- incluido o campo `Motivo da alteracao` no modal `Atualizar prazos da OS`;
- incluida a area de `Autorizacao administrativa` quando o backend exigir aprovacao de administrador;
- ajustado o JavaScript para enviar motivo, datas auxiliares e credenciais no AJAX;
- reforcado o backend com fallback para as datas atuais da OS.

## Arquivos principais

- `app/Views/os/index.php`
- `public/assets/js/os-list-filters.js`
- `app/Controllers/Os.php`
- `app/Config/SystemRelease.php`

## Observacoes

- o motivo informado passa a alimentar o log operacional da OS;
- o modal continua usando SweetAlert2 para avisos e erros do fluxo.
