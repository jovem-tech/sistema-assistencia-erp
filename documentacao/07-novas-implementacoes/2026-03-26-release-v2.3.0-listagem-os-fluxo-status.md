# Release v2.3.0 - Listagem operacional de OS + workflow configuravel de status

Data: 26/03/2026

## Resumo

Esta release promove a listagem de Ordens de Servico a uma grade operacional mais rica e introduz uma tela administrativa para configurar o fluxo de status da OS.

## Entregas principais

### 1. Nova leitura operacional da listagem `/os`
- coluna `Equipamento` reestruturada para exibir:
  - `Tipo`
  - `Marca`
  - `Modelo`
- coluna `Relato` com quebra de linha e ajuste de fonte para preservar ate 4 linhas uteis
- coluna `Datas` substituindo `Data Abertura`, com:
  - `Entrada`
  - `Prazo`
  - `Entrega`
- indicadores visuais de prazo:
  - verde
  - amarelo
  - laranja
  - vermelho

### 2. Troca de status direto da listagem
- o badge de status agora e clicavel
- a listagem abre modal com destinos permitidos por OS
- a troca acontece por AJAX, sem reload completo da pagina

### 3. Workflow configuravel de status
- nova tela `Gestao de Conhecimento -> Fluxo de Trabalho OS`
- configuracao de:
  - ordem dos status
  - status ativo
  - status final
  - status de pausa
  - destinos permitidos por etapa

### 4. Regra de fallback
- se nao existirem transicoes personalizadas salvas, o sistema usa fallback automatico por `ordem_fluxo`
- nesse modo, cada status ativo pode avancar para o proximo ou retornar para o anterior
