# Nota de Implementacao: Release v2.2.13 - Obrigatoriedade de Identificacao Visual

## Contexto
Para evitar equipamentos cadastrados com informacoes genericas ou fotos ausentes (que dificultam a identificacao e triagem), a versao `2.2.13` implementa a validacao obrigatoria de **Cor** e **Foto de Perfil** no fluxo de cadastro rapido de equipamentos dentro da Ordem de Servico.

## Alteracoes Principais

### 1. UX: Redirecionamento por Abas
O modal de novo equipamento agora possui uma validacao em cascata no evento de clique do botao "Cadastrar":
- Se campos de **Info** (Tipo, Marca, Modelo) estiverem ausentes, o sistema volta para a aba Info.
- Se a **Cor** nao foi selecionada, o sistema muda para a aba Cor e abre o seletor.
- Se nenhuma **Foto** foi adicionada, o sistema muda para a aba Foto.
- O alerta visual e dado via `Swal.fire` (SweetAlert2) antes de qualquer submissao AJAX.

### 2. Mudanca de Estado Inicial
- O cadastro de novo equipamento nao inicia mais com a cor "Preto" selecionada por padrao.
- O estado inicial agora e "Cor nao selecionada" (`hex: ''`), forçando o usuario a identificar visualmente o aparelho antes de salvar.
- `updateColorUIOS` foi refatorada para suportar este estado "vazio" com placeholders visuais neutros.

### 3. Integridade de Dados
- A validacao impede a criacao de registros "fantasmas" sem identificacao visual.
- Garante que a galeria principal da OS e o card do equipamento tenham pelo menos uma imagem de referencia.

## Arquivos Afetados
- `app/Views/os/form.php`: Reestruturacao do handler `btnSalvarNovoEquip` e melhoria do `updateColorUIOS`.
- `app/Config/SystemRelease.php`: Incremento para `2.2.13`.
- `documentacao/07-novas-implementacoes/historico-de-versoes.md`: Registro da release.

## Como Testar
1. Abra a tela de Nova OS.
2. Selecione um cliente e clique em `+ Novo` equipamento.
3. Preencha apenas a aba Info e tente salvar. O sistema deve barrar e levar para a aba Cor.
4. Escolha uma cor e tente salvar. O sistema deve barrar e levar para a aba Foto.
5. Tire ou selecione uma foto e o salvamento sera permitido.
