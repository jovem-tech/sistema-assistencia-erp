# Manual do Usuário — Perfil

## 📋 Visão Geral

A área **Meu Perfil** permite atualizar seus dados pessoais e senha sem depender do administrador.

---

## 🧭 Navegação
**Caminho:** Menu superior → Perfil

---

## ✏️ Atualizar Dados

Você pode editar:
- Nome completo
- Email
- Telefone
- Foto de perfil

---

## 🔒 Alterar Senha

Para alterar a senha:
1. Informe a senha atual
2. Digite a nova senha
3. Confirme a nova senha

Se os campos de senha estiverem vazios, nada é alterado.
