# Release v2.10.6 - Central de Mensagens: composer com altura compacta forcada

Data: 02/04/2026

## Objetivo

Eliminar de forma mais agressiva o estado em que o `textarea` do composer permanece alto mesmo vazio, desalinhando da acao principal de envio.

## O que mudou

- o `textarea` recebeu altura compacta com prioridade alta no CSS para manter alinhamento visual com o botao `Enviar`;
- o resize do composer agora limpa estados vazios de forma mais defensiva, inclusive quando sobra `height` inline de interacoes anteriores;
- a Central reaplica o ajuste de altura apos o bootstrap e apos abrir uma conversa;
- o breakpoint `<=360px` ganhou o mesmo comportamento compacto aplicado ao desktop e ao `<=320px`.

## Arquivos impactados

- `public/assets/css/design-system/layouts/central-mensagens.css`
- `public/assets/js/central-mensagens.js`
- `app/Config/SystemRelease.php`
- `documentacao/06-modulos-do-sistema/central-de-mensagens.md`
- `documentacao/07-novas-implementacoes/historico-de-versoes.md`
- `documentacao/08-correcoes/2026-04-02-central-mensagens-composer-altura-estavel.md`
- `documentacao/README.md`

## Resultado esperado

- composer vazio no mesmo patamar visual do botao de envio;
- expansao apenas quando houver conteudo multi-linha real;
- menor recorrencia do campo "preso" em altura antiga apos mudar de conversa ou voltar para a Central.
