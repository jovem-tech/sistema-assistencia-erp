# Manual do Administrador — Configuração do Sistema

## ⚙️ Dados da Empresa

**Caminho:** Configurações → Dados da Empresa

Configure as informações que aparecem nas impressões de OS e no topo do sistema:

| Campo | Descrição |
|-------|-----------|
| Nome do Sistema | Nome exibido no navegador e sidebar |
| Razão Social | Nome legal da empresa |
| CNPJ | Documento da empresa |
| Telefone | Contato da empresa |
| Endereço | Endereço para impressões |
| Logo | Imagem exibida no topo das impressões |
| Ícone (favicon) | Ícone da aba do navegador |
| Cor Primária | Cor principal do tema |
| Tema Padrão | Dark ou Light |
| Mensagem de Rodapé | Texto no final das OS impressas |
| Garantia Padrão (dias) | Dias de garantia padrão de todos os reparos |

---

## 🎨 Temas

O sistema suporta dois temas:
- **Dark Mode** (padrão) — Glassmorphism escuro
- **Light Mode** — Interface clara

O usuário pode alternar pelo ícone 🌙/☀️ no canto superior direito. A preferência é salva por usuário via `localStorage`.
