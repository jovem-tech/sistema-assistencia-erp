# Checklists de Entrega

## Toda nova feature do app deve validar

1. fluxo principal funcionando;
2. estados de loading, erro e vazio;
3. responsividade 360px e 320px;
4. reatividade sem refresh;
5. documentacao atualizada;
6. compatibilidade com o ERP confirmada.

## Feature com fotos deve validar

1. camera;
2. galeria;
3. crop;
4. preview;
5. exclusao;
6. persistencia correta;
7. caminho correto em `public/uploads`.

## Feature com notificacao deve validar

1. permissao;
2. service worker;
3. subscription;
4. entrega;
5. fallback;
6. diagnostico visual.
