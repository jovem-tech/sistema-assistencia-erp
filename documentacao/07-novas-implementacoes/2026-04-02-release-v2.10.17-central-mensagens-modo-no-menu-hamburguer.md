# Release v2.10.17 - Central de Mensagens: modo de atendimento no menu hamburguer

Data: 02/04/2026

Escopo: `Central de Mensagens` (`/atendimento-whatsapp`)

## Objetivo

Manter o cabecalho da thread mais limpo, movendo os controles de modo de atendimento para dentro do menu hamburguer.

## Implementacao

1. `Bot ativo/Bot desativado` movido para o menu hamburguer.
2. `Aguardando atendimento humano` movido para o menu hamburguer.
3. `Status` e `Prioridade` permaneceram visiveis no cabecalho.
4. `Ocultar/Mostrar contexto` permanece visivel ao lado do hamburguer.
5. Reforco visual no dropdown para os estados de modo (ativo/inativo).
