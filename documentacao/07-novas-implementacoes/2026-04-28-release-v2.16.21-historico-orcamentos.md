# Release 2.16.21 - Historico de Status dos Orcamentos

Data: 28/04/2026  
Versao ERP: `2.16.21`

## Objetivo

Restabelecer a exibicao do timeline de status do orcamento na tela de visualizacao, mantendo a rastreabilidade operacional acessivel no proprio modulo.

## Entrega funcional

- corrigido o mapeamento da variavel de historico em `app/Views/orcamentos/show.php`;
- o card `Historico de status` volta a renderizar os eventos retornados por `OrcamentoStatusHistoricoModel::timeline()`;
- a tela deixa de exibir `Sem historico de status` quando o backend possui registros reais para o orcamento.

## Arquivos principais

- `app/Views/orcamentos/show.php`
- `app/Config/SystemRelease.php`

- `documentacao/07-novas-implementacoes/historico-de-versoes.md`
