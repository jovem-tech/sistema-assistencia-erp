# Release v2.8.0 - OS legado com filtro dedicado e busca global

Data: 28/03/2026

## Resumo
Esta release melhorou a operacao de conferência das ordens migradas do sistema antigo sem criar uma tela paralela. A abordagem adotada foi:

- manter a listagem principal em `/os`
- adicionar um seletor visual de origem com o atalho `Somente legado`
- ampliar a busca global da navbar para localizar `numero_os_legado`

## O que mudou

### 1. Filtro dedicado para OS legado na listagem
- A tela `/os` ganhou a barra `Origem das ordens`.
- O operador pode alternar entre:
  - `Todas as OS`
  - `Somente legado`
- O filtro funciona por AJAX, sem reload manual.
- O estado do filtro passa a persistir em:
  - URL (`legado=1`)
  - localStorage
  - chips de filtros ativos
  - payload do DataTable server-side

### 2. Busca global da navbar por numero legado
- A busca superior do sistema agora tambem consulta `os.numero_os_legado`.
- Quando a OS for legada, o resultado contextual passa a exibir `Legado: <numero_antigo>` no subtitulo.
- O catalogo interno da busca rapida ganhou a entrada `OS Legado`, apontando para `/os?legado=1`.

### 3. UX e responsividade
- O seletor de origem foi integrado ao design system da listagem de OS.
- Em tablet e mobile, os botoes reorganizam para ocupar a largura disponivel sem quebrar layout.
- O comportamento permanece consistente com os filtros premium da pagina.
