# Tokens Visuais

Atualizado em 04/04/2026.

## Cores base

- fundo externo: `#eef2f7`
- card branco: `#ffffff`
- texto principal: `#0f172a`
- texto secundario: `#64748b`
- borda leve: `#dfe7f2`
- header gradient: `#0f172a -> #1e3a8a`

## Feedback

- status/info: azul claro
- prioridade: amarelo suave
- erro: vermelho funcional
- sucesso: verde funcional

## Tipografia

- familia base: `"Segoe UI", "SF Pro Text", -apple-system, BlinkMacSystemFont, "Helvetica Neue", Arial, sans-serif`
- titulo principal: 20px
- subtitulo: 13px
- labels auxiliares: 12px

## Raios

- card: `16px`
- controles principais: `12px`
- chips: `999px`

## Sombras

- cards: sombra suave curta
- overlays: sombra mais profunda

## Motion

- hover leve
- transicoes curtas
- sem animacoes longas ou exibicionistas
