# 2026-04-28 - Release v2.16.17

## Resumo

Patch de impressao termica da OS para transformar `Bobina 80mm` em um fluxo direto de impressao do sistema operacional, sem abrir modal intermediario.

## Versao oficial

- arquivo fonte da release: `app/Config/SystemRelease.php`
- versao anterior: `2.16.16`
- versao atual: `2.16.17`

## Escopo consolidado nesta release

- o dropdown `Imprimir` passou a abrir `Bobina 80mm` diretamente no documento termico;
- o endpoint de impressao passou a aceitar `auto_print=1` para disparar `window.print()` automaticamente;
- `app/Views/os/print.php` ganhou um layout termico dedicado, separado da estrutura visual do `A4`;
- o modelo termico organiza em leitura linear as secoes de identificacao, tecnico, cliente, equipamento, relato, diagnostico, checklist, financeiro e orcamento.

## Arquivos-chave

- `app/Config/SystemRelease.php`
- `app/Controllers/Os.php`
- `app/Views/os/show.php`
- `app/Views/os/print.php`
- `documentacao/06-modulos-do-sistema/ordens-de-servico.md`
- `documentacao/07-novas-implementacoes/historico-de-versoes.md`

## Observacao

Se existir override manual por banco em `configuracoes.sistema_versao`, ele deve ser alinhado para `2.16.17` para manter login e rodape consistentes com a release de codigo.
