# Release v2.9.6 - Menuia com validacao real e badge confiavel

Data: 31/03/2026

## Objetivo

Tornar a integracao com a Menuia previsivel no painel de configuracoes, com:
- URL canonica
- validacao real das credenciais
- feedback visual coerente entre sucesso, erro e credencial nao validada

## O que mudou

### 1. URL canonica da Menuia
- o sistema passou a tratar `https://chatbot.menuia.com/api` como endpoint oficial
- se o operador salvar `https://api.menuia.com/api`, a URL e normalizada automaticamente

### 2. Teste de conexao sem falso negativo
- o botao `Testar conexao` passou a enviar uma mensagem unica com timestamp para o telefone de teste
- isso evita falha por mensagem repetida quando a Menuia rejeita payload duplicado

### 3. Badge de status confiavel
- a tela `Configuracoes -> Integracoes` agora diferencia:
  - `Menuia conectada`
  - `Erro Menuia`
  - `Menuia nao validada`
- o status salvo so e considerado valido quando corresponde exatamente ao conjunto atual de:
  - URL
  - Appkey
  - Authkey

### 4. Invalidacao de status ao trocar credenciais
- qualquer troca de URL, Appkey ou Authkey limpa o ultimo status salvo
- isso impede que a tela continue mostrando sucesso de uma validacao antiga

## Impacto operacional
- o administrador consegue confiar no badge da Menuia sem precisar abrir console ou testar manualmente por fora
- ao trocar as credenciais, a tela volta para estado de revisao ate um novo teste ser executado
